# Chico Romelo — Official Website

Official website for the band Chico Romelo, designed and developed with a strong focus on performance, responsiveness, accessibility, and modern web standards.

Built using HTML5, CSS3, and Vanilla JavaScript, the platform delivers a lightweight and fast user experience while supporting rich multimedia content and interactive features tailored for music promotion and audience engagement.

## Features

- Fully responsive layout using Flexbox
- Interactive modal system
- Contact form integration with Formspree
- Embedded YouTube video content
- SEO-focused architecture optimized for music artist discovery
- WebP image optimization and lazy loading
- Smooth animations and transitions
- Downloadable media and file management
- Semantic and accessible HTML structure
- Optimized loading performance across desktop and mobile devices

## Performance & SEO

The website was carefully engineered to achieve fast loading times and improved search engine visibility through:

- Lightweight Vanilla JavaScript architecture
- Optimized asset delivery
- Semantic HTML5 markup
- Responsive-first design approach
- Lazy loading strategies
- Optimized media formats
- SEO best practices for bands and music-related content

## Local Development

The site uses clean URLs (`/sobre`, `/lancamentos`, etc.) that match how GitHub Pages serves the files. To replicate this locally, use [`npx serve`](https://www.npmjs.com/package/serve) instead of Live Server:

```bash
npx serve .
```

The included `serve.json` enables `cleanUrls` automatically, so `/sobre` resolves to `sobre.html` exactly as it does on production. GitHub Pages ignores `serve.json`.
